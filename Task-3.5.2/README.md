### Задание

Используя однократный вызов свертки, реализуйте функцию `evenOnly`, которая выбрасывает из списка элементы, стоящие на нечетных местах, оставляя только четные.

```
GHCi> evenOnly [1..10]
[2,4,6,8,10]
GHCi> evenOnly ['a'..'z']
"bdfhjlnprtvxz"
```

### Ответ

```
evenOnly :: [a] -> [a]
evenOnly = map snd . filter (even . fst) . zip [1..]
```
